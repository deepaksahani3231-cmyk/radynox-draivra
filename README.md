# Radynox-Draivra

Radynox-Draivra is an Astro + Tailwind website focused on practical founder support, operations clarity, and sustainable growth.

## Tech

- Astro v6
- Tailwind CSS v4
- TypeScript
- MDX

## Commands

- npm run dev
- npm run build
- npm run preview
- npm run check
- npm run fix

## Project Structure

- src/config.yaml: site and SEO config
- src/navigation.ts: header and footer links
- src/pages: route definitions
- src/components: UI and widgets
- src/data/post: blog content
- vendor/integration: config virtual module integration

## Notes

- Virtual config module: radynox-draivra:config
- Contact route: /contact
- Join route alias: /join -> /join-mission
